Question-3:what is event delegation in the DOM?

Event delegation is a technique in JavaScript where a single event listener is added to a parent element instead of multiple event listeners for individual child elements.

Question-4:what is the event bubbling in the DOM?

Event bubbling is a concept in the DOM where an event that occurs on a child element bubbles up to its parent elements.
